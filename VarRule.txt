Some variables are standardized and so you'll see this format:

<TypeVariable>_<TypeReturn><NameVariable> Ex: txt_bCreate  

List of possible names:

Type Variable:
scr (screen)
str (string)
stm (stream)
txt (files)
gl (opengl)
form (windows form)
time (time)
err (error)
log (logs)
ncs (ncscript)
msgb (messagebox)
ex (exception)
res (resource files)
net (network)
chk (check)
xin (xinput)
mtx (matrix)
cam (camera)

Type Return:
b (bool)
i (int)
f (float)
s (string)
c (generic class)
r (resource file)
e (exception)
n (enumeration)
a (array)
l (list)
m (matrix)
t (time)
p (point2d/point3d)
w (windows class)
x (input)
y (byte or unsigned char)
u (unsigned int)
<nothing> (funcion void)

Name Variable is a short description about the variable
